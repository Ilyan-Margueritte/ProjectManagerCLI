Fonctionnalités clés

Initialisation Rapide

Créez des projets à partir de templates préconfigurés en quelques secondes.

Gestion des Dépendances

Intégration automatique avec npm, yarn, pip, cargo et autres gestionnaires de paquets.

Templates Personnalisables

Créez et partagez vos propres templates pour standardiser vos projets.

Automatisation

Scripts de build, test et déploiement configurables pour chaque projet.

Collaboration

Synchronisation des configurations d'équipe et partage de templates.

Outils Intégrés

Linters, formatters et hooks git préconfigurés selon vos standards.

Installation

Via NPM

npm install -g projectmanager-cli

Via Yarn

yarn global add projectmanager-cli

Via Cargo (Rust)

cargo install projectmanager-cli

Comment l'utiliser

Créer un nouveau projet

# Créer un projet avec un template spécifique pmcli init mon-projet --template node-ts # Ou interactivement pmcli init

Gérer les templates

# Lister les templates disponibles pmcli template list # Ajouter un template personnalisé pmcli template add ./mon-template --name custom-template

Exécuter des tâches

# Lancer une tâche définie dans pmcli.json pmcli run build # Ou lancer en mode watch pmcli run dev --watch

Configuration

// pmcli.json { "name": "mon-projet", "version": "1.0.0", "template": "node-ts", "scripts": { "build": "tsc", "test": "jest", "dev": "nodemon src/index.ts" } }

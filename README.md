# PMCLI — Project Manager CLI

PMCLI est un outil de ligne de commande simple et puissant pour gérer vos projets de développement localement.

## Installation

```bash
pip install pmcli
```

## Fonctionnalités

- 🚀 **Initialisation rapide** : Crée un projet et initialise Git automatiquement.
- 📦 **Gestion des tâches** : Ajoutez, listez et terminez des tâches par projet.
- 📊 **Suivi de progression** : Visualisez l'avancement avec des barres de progression.
- 📂 **Accès rapide** : Ouvrez vos dossiers de projet en une commande.

## Utilisation

```bash
pmcli init mon-projet
pmcli add-task mon-projet "Coder l'API"
pmcli list
pmcli done mon-projet "Coder l'API"
```
